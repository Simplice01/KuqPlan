KuqPlan – Plateforme Laravel de Rencontres avec Système d’Achat de Contacts

KuqPlan est une plateforme web développée avec Laravel permettant aux utilisateurs de consulter des profils et d’acheter des points afin d’accéder aux coordonnées des personnes qui les intéressent.
Le système repose sur un modèle simple et efficace : l’utilisateur explore les profils, achète des points, puis dépense ces points pour révéler les informations de contact.

Ce projet met en avant des compétences en Laravel, Blade, gestion d’authentification, logique de paiement interne et organisation MVC.
